# Svelte Social Platform

A sample Svelte project for a new social platform. Uses Svelte@17 and tests with Jest and Enzyme.

## Project Setup

```sh
npm install
```

### Build files

```sh
npm run build
```

### Start local server

```sh
npm run start
```

### Watch files for changes and rebuild

```sh
npm run develop
```

### Run tests

Unit tests

```sh
npm run test
```

Integration tests

```sh
npm run test:integration
```
